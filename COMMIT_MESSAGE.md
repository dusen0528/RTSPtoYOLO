perf: 성능 최적화 및 에러 처리 개선

주요 변경사항:

성능 최적화:
- 프레임 스킵 최적화: 3프레임마다 1번만 YOLO 추론 실행 (연산량 60-70% 감소)
- 모자이크 방식 개선: ratio=0.05로 축소/확대하여 블러 연산 속도 향상
- yolov8n-face.pt 모델 자동 감지 및 우선 사용 (얼굴 전용 모델)
- skip_interval 설정 추가 (기본값: 3)

에러 처리 개선:
- 구체적인 에러 메시지 반환 (RTSP 연결 실패 원인 분석)
- 자동 재시도 로직 개선 (지수 백오프: 1초, 2초, 3초, 4초, 5초)
- 프레임 수신 중단 자동 감지 및 재연결 (5초 이상 프레임 없음)
- ERROR 상태에서도 재시작 가능하도록 수정
- FFmpeg 에러 타입별 구체적인 메시지 (연결 실패, 타임아웃, 권한 오류)

로그 개선:
- FFmpeg H.264 디코딩 경고 메시지 필터링 (불필요한 로그 제거)
- 프로젝트 전반 이모지 제거

설정 변경:
- config.py에 skip_interval 추가 (BLUR_SKIP_INTERVAL 환경 변수로 설정 가능)
